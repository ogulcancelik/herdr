## 选择你的路径

<CardGrid>
  <Card title="第一次接触终端复用器？">
    上手不需要学习快捷键。Herdr 以鼠标为先：点击 Pane、拖动边框，通过右键菜单分割和切换。

    [快速开始 →](/zh-cn/docs/quick-start/)
  </Card>
  <Card title="从 tmux 或 zellij 迁移过来？">
    这套模型你已经很熟悉了。前缀键是 `ctrl+b`，Pane 会持久保留，分离和重新连接的行为都符合你的预期。

    [核心概念 →](/zh-cn/docs/concepts/) · [按键绑定 →](/zh-cn/docs/configuration/#按键绑定)
  </Card>
</CardGrid>

## 或者让你的 Agent 来介绍

已经在用 AI 编程 Agent？让它来帮你入门。粘贴这个提示词：

```text
Help me understand and set up Herdr. Read https://herdr.dev/agent-guide.md first, then walk me through it step by step.
```

这份指南会教你的 Agent Herdr 的概念、安装、配置和常见问题修复方法，让它的回答保持准确，而不是即兴发挥。
