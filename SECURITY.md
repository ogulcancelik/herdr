# Security Policy

## Supported Versions

Only the latest stable release builds are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.7  | :white_check_mark: |
| < 0.7   | :x:                |

## Reporting a Vulnerability

We take the security of `herdr` seriously. If you find a security vulnerability, please do not disclose it publicly. Instead, report it privately using GitHub's private vulnerability reporting feature under the **Security** tab of this repository.

Please provide a clear description of the vulnerability, reproduction steps, and potential impact. We will coordinate a resolution and release a security advisory promptly.
